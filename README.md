# Kerala Fat Loss Diet Plan Generator

A modern web application that generates a personalized Kerala-style fat loss diet plan based on user inputs such as age, gender, weight, height, activity level, and food preference.

The application calculates daily calorie requirements and automatically generates a weekly meal plan along with a grocery shopping list.

---

## Features

- Personalized calorie calculation using BMR and TDEE formulas
- Weekly meal plan generator
- Kerala-style food recommendations
- Vegetarian and Non-Vegetarian options
- Automatic grocery list generation
- Water intake recommendation
- Dark mode support
- Modern glassmorphism UI

## How It Works

1. User enters personal information:
   - Age
   - Gender
   - Weight
   - Height
   - Activity level
   - Food preference

2. The system calculates:

   - **BMR (Basal Metabolic Rate)**
   - **TDEE (Total Daily Energy Expenditure)**

3. Based on the selected weight loss goal, the app:

   - Calculates calorie deficit
   - Generates a weekly meal plan
   - Creates a grocery shopping list
